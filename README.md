# Bingo4D
Bingo testing

## Instructions

1. Clone the repo.

2. Execute npm install inside (you will need NodeJs installed)

3. Execute npm start.

Just press on start and the bingo game will start!

## Important

It is all in Spanish, but you can easily modify the code to make it in any other language :)

Enjoy!
